🔧 Google Sheet Setup Guide

One-time setup · Takes 5 minutes · Enables auto-submit for all SHAPE forms
1 Create a Google Sheet
  1. Open sheets.google.com
  2. Click + Blank spreadsheet
  3. Rename it to SHAPE Client Data (click on "Untitled spreadsheet" at the top)
✅ You now have an empty Google Sheet. Don't add any data — the script will auto-create headers.
2 Open Apps Script
  1. In your Google Sheet, click Extensions menu (top menu bar)
  2. Click Apps Script
  3. A new tab opens with a code editor
⚠️ You'll see some default code like function myFunction() {} — DELETE ALL OF IT.
3 Paste the SHAPE Script
  1. Select all the default code (Ctrl+A) and delete it
  2. Open the file SHAPE_Google_Apps_Script.js (downloaded with this guide)
  3. Copy ALL the code from that file
  4. Paste it into the Apps Script editor
  5. Click 💾 Save (or Ctrl+S)
  6. When asked for a project name, enter: SHAPE Form Handler
✅ The script handles both pre-mentorship and weekly check-in submissions. It also sends you an email notification for each new submission!
4 Deploy as Web App
  1. Click the Deploy button (top right, blue button)
  2. Click New deployment
  3. Click the ⚙️ gear icon next to "Select type"
  4. Choose Web app
  5. Fill in:
    • Description: SHAPE Forms
    • Execute as: Me
    • Who has access: Anyone
  6. Click Deploy
⚠️ First time? Google will ask you to authorize:
  1. Click Authorize access
  2. Choose your Google account
  3. You'll see a warning "Google hasn't verified this app" — click Advanced
  4. Click Go to SHAPE Form Handler (unsafe)
  5. Click Allow
This is safe — you're authorizing YOUR OWN script to write to YOUR OWN sheet.
5 Copy the Web App URL
  1. After deployment, you'll see a Web app URL
  2. It looks like this:
https://script.google.com/macros/s/AKfycbx...long-string.../exec
  1. Click Copy to copy the URL
⚠️ IMPORTANT: Copy the FULL URL. It's very long — make sure you get all of it.
6 Paste in SHAPE CRM
  1. Open SHAPE CRM (the HTML file)
  2. Go to Roster page (main page)
  3. Scroll down to 👨‍🏫 Coach Profile
  4. Paste the URL in the "Google Sheet URL" field
  5. You should see: ✅ Auto-submit ON
✅ DONE! All forms will now auto-submit to your Google Sheet.
7 Test It!
  1. In SHAPE CRM, download the Onboarding Form
  2. Open the downloaded form in your browser
  3. Fill in some test data (name: "Test Client", phone: "1234567890", weight: 80, height: 175, age: 30)
  4. Tap ✅ Submit
  5. Open your Google Sheet — you should see a new row! 🎉
✅ If you see the row, everything works! Now do the same test with a Check-In form.
❌ If no row appears:
  • Make sure you copied the FULL URL (it's very long)
  • Make sure "Who has access" is set to "Anyone"
  • Try re-deploying: Deploy → Manage deployments → edit → redeploy
🎁 Bonus: Email Notifications
The script automatically sends you an email notification whenever:
  • 🆕 A new client submits the Pre-Mentorship form — you get their name, phone, weight, goal, and the import code
  • 📏 A client submits a Weekly Check-In — you get their weight, measurements, and the import code
You don't even need to check the Google Sheet! The email has the code — just paste it in SHAPE CRM → Import.
